Welcome to polyrat's documentation!
===================================

*polyrat* is a Python package for working with polynomials and rational functions.
This includes features like

* a variety of polynomial bases Legendre, Lagrange, Arnoldi,...
* computing roots for polynomials in any basis
* constructing polynomial approximations that are optimal in a variety of norms
* constructing rational approximations
